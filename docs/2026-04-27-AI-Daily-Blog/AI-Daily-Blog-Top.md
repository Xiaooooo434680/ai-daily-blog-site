# Dario Amodei、AI 炒作、AI 安全与 vibe-coding 灾难

**作者：Gary Marcus**

AI 行业最擅长的事情之一，就是在公开场合大谈安全的重要性，同时在实际操作中不断降低安全标准。

Anthropic 的 CEO Dario Amodei 一直将自己定位为负责任的 AI 发展代言人。他频繁出现在国会听证会上，发表关于 AI 存在性风险的长篇论文，并一再强调 Anthropic 的"宪法 AI"方法比竞争对手更安全。

但在实践中，我们看到的是另一番景象。Anthropic 与其他 AI 公司一样，在市场竞争的压力下不断加速发布产品。Claude 模型的每一次迭代都在追求更强的能力，而安全评估往往在发布之后才完成——而不是在发布之前。

与此同时，一个更令人担忧的趋势正在兴起：vibe-coding。这个词描述的是非专业开发者使用 AI 编码工具（如 Claude Code、GitHub Copilot 等）生成代码，然后不经审查就直接部署到生产环境的行为。

Vibe-coding 的问题不在于工具本身——AI 编码助手确实能提高效率——而在于使用方式。当不具备安全意识的用户盲目信任 AI 生成的代码时，结果就是安全漏洞、数据泄露和系统故障的激增。我看到的案例包括：AI 生成的 SQL 查询导致数据库被清空、未经审查的 API 端点暴露敏感数据、以及硬编码的凭证被提交到公开仓库。

AI 布道者不愿意谈论这些失败案例。他们更喜欢展示那些令人印象深刻的演示——AI 在几秒钟内生成了完整的应用程序。但他们从不展示这些应用程序在生产环境中崩溃的后续。

Dario Amodei 和他的同行需要停下来说实话：AI 工具是强大的，但它们也是危险的。安全不能是事后才考虑的——它必须从一开始就内建在产品中。否则，我们正在用越来越快的速度制造越来越大的问题。

# 追踪 OpenAI 微软已失效的 AGI 条款

**作者：Simon Willison**

多年来，微软与 OpenAI 的合作关系中包含一个奇特而重要的条款：一旦 OpenAI 董事会认定已实现通用人工智能（AGI），微软对该技术的所有商业知识产权权利将立即终止。

这个条款最初设计为一个安全阀——理论上，它可以防止任何一家公司垄断 AGI 技术。如果 OpenAI 真的创造出了 AGI，它将不属于微软，而是属于全人类——至少在 OpenAI 的非营利使命宣言中是这么说的。

但这个条款从一开始就充满模糊性。谁来定义 AGI？OpenAI 的董事会如何构成？如果董事会成员与微软有利益关系怎么办？这些问题的答案随着时间的推移变得越来越复杂。

2023 年底，在 Sam Altman 被短暂罢免又复职的事件中，这个条款成为了焦点。OpenAI 的非营利董事会几乎触发了一个连锁反应——如果董事会结构发生变化，AGI 条款的执行机制也会随之改变。

到了 2025 年，随着 OpenAI 转型为公益公司（PBC），这个条款被正式从协议中移除。微软现在拥有 OpenAI 技术的永久商业权利，无论是否实现 AGI。更重要的是，OpenAI 的治理结构已经发生了根本性变化，最初设想中的"非营利董事会认定 AGI"机制已不复存在。

这条 AGI 条款的历史弧线——从一个理想主义的安全阀到最终被商业现实碾碎——完美地概括了 AI 行业过去十年的演变。

# Xerox 如何发明 GUI 又失去它

Xerox 是一家那些想显得聪明或有远见的人都应该了解的公司。在 1960 年代，Xerox 似乎是一家注定永远统治办公设备市场的公司。它的复印机业务如此成功，以至于公司有了大笔资金投入前瞻性研究。

于是，Xerox 帕洛阿尔托研究中心（PARC）诞生了。

1970 年代，PARC 聚集了当时世界上最聪明的计算机科学家。在这里，他们发明了图形用户界面（GUI）、鼠标、以太网、激光打印机、面向对象编程——几乎现代个人计算的每一个基础概念都可以追溯到 PARC 的走廊。

但 Xerox 未能将这些发明商业化。

原因很多。首先，Xerox 的公司 DNA 是复印机业务。高管们衡量一切的标准是对复印机销售的贡献——个人计算看起来像一个奇怪的、不相关的小玩意儿。其次，PARC 位于西海岸，距离东海岸总部有数千英里，文化隔离导致总部无法理解实验室的价值。第三，当 Xerox 终于尝试推出 Xerox Star 工作站时，定价高达 16,000 美元，远超市场接受能力。

1979 年，Steve Jobs 参观了 PARC。他看到了 GUI 的演示，立刻理解了它的革命性意义。正如他后来所说："我看到图形用户界面的那一刻，就知道这是每台个人计算机的未来。"Jobs 将这个概念带回 Apple，先是 Lisa，然后是 Macintosh。

Bill Gates 紧随其后，Microsoft Windows 的推出完成了 GUI 从 Xerox 实验室到全球桌面的旅程。Xerox PARC 的科学家们发明了一切，但收获商业成功的却是别人。

这个故事对今天 AI 行业的启示是显而易见的：拥有最好的技术不意味着你最终会赢。商业化能力、时机把握和市场理解同样重要。
