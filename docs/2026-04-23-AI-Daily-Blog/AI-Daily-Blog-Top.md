# 独家：微软将于 6 月将所有 GitHub Copilot 订阅用户转向 Token 计费

一份内部文件揭示了微软关于 GitHub Copilot 定价的重大改革计划。根据这份泄露的文件，微软将于今年 6 月全面启动 Token 计费模式，彻底改变 Copilot 的商业模式。

**新的定价结构**

对于 Copilot Business 用户：
- 费用：$19 美元/人/月
- 包含：$30 的池化 AI 积分
- 超出部分按实际消耗计费

对于 Copilot Enterprise 用户：
- 费用：$39 美元/人/月
- 包含：$70 的池化 AI 积分
- 同样按超出部分计费

这意味着什么？这意味着结束"一个价格、无限使用"的传统订阅模式。取而代之的是基于实际使用量的动态计费。

**Token 计费的影响**

Token 计费模式对用户有两方面影响：

**正面影响**：轻度用户可能从新模式中受益。如果你不经常使用 Copilot，实际消耗可能低于原来的固定费用。

**负面影响**：重度用户可能面临更高账单。AI 编程工具的价值在于高频使用，而高频使用意味着高 Token 消耗。

这对企业采购尤其重要。以前，IT 部门可以预测每个席位的固定成本。现在，成本将变得不可预测，取决于开发者的使用模式。

**行业趋势**

微软这一举动反映了一个更大的行业趋势：AI 产品正在从"按用户数"向"按使用量"转变。这与云计算的演进路径一致——最初也是按虚拟机数收费，后来转向按实际计算资源计费。

对于 AI 行业来说，问题是：**开发者能否接受这种不确定性？**

# Claude 帮助 Firefox 发现 271 个安全漏洞

Mozilla 与 Anthropic 的合作取得了惊人成果。作为持续合作的一部分，Mozilla 获得了 Claude Mythos Preview 早期版本的访问权限，并将其应用于 Firefox 安全审计。

本周发布的 Firefox 150 包含了这次 AI 驱动安全审计的成果——**271 个安全漏洞修复**。

这个数字令人震惊。传统的安全审计通常会发现十几个到几十个漏洞，而 AI 审计一次性发现了 271 个。这说明什么？

**AI 在安全领域的潜力**

AI 系统在发现漏洞方面有几个独特优势：

**1. 覆盖面广**
AI 可以系统地扫描代码库的每一行，而人类审计员必须有所取舍。

**2. 速度快**
AI 可以在数小时内完成人类安全团队数周的工作量。

**3. 发现隐藏模式**
AI 擅长发现人类可能忽略的微妙模式——特别是那些涉及复杂交互的安全问题。

**对软件安全的启示**

这次实验揭示了几个重要启示：

首先，**AI 不会替代人类安全专家，而是放大他们的能力**。Mozilla 团队可以使用 AI 工具快速扫描代码，然后专注于分析和修复最重要的发现。

其次，**开源项目可以从 AI 安全合作中获益良多**。Firefox 作为开源浏览器，其代码库对安全社区透明，现在也受益于 AI 审计能力。

第三，**AI 公司的安全承诺应该用实际成果来验证**。Mozilla-Anthropic 合作模式证明，通过深度集成，AI 公司可以直接为使用其技术的项目创造安全价值。

# 卢德派与 AI 数据中心

Is it time to start burning down datacenters?（是时候开始烧毁数据中心了吗？）

Sean Goedecke 在这篇文章中探讨了一个越来越普遍的情绪：反 AI 数据中心运动。

**事件背景**

最近发生了几起引发广泛关注的事件：

一个 Indianapolis 市议会成员因其支持数据中心的立场而住所被枪击。Sam Altman（OpenAI CEO）的家被纵火。这些极端行为的背后，是对 AI 数据中心的强烈反对情绪。

**什么是卢德派？**

历史上的卢德派（Luddites）是 19 世纪初英国的纺织工人，他们摧毁机器以抗议自动化带来的就业威胁。这个名字后来成为"技术恐惧者"的代名词。

但 Goedecke 指出，对卢德派的简单嘲笑忽略了一个重要事实：**他们的担忧往往是有道理的**。自动化确实会造成某些工作的消失，即使它总体上创造了更多财富。

**反数据中心运动的不同层面**

当前的反数据中心情绪包含几个不同层面：

**1. 环境担忧**
数据中心消耗大量电力。在可再生能源尚未完全普及的情况下，新建数据中心确实会增加碳排放。

**2. 社区影响**
大型数据中心需要大量土地、水资源（用于冷却），并推动房价上涨。当地居民有合理担忧。

**3. AI 本身的不确定性**
这是最难讨论的一点。AI 是否会取代人类工作？AI 是否安全？这些问题没有简单答案。

**我们应该如何回应？**

Goedecke 认为，简单地将反对者标签化为"卢德派"或"不了解技术"是不公平的。他们的一些担忧是合理的，值得认真对待。

但同样正确的是，**技术进步本身不是问题，问题在于进步的果实如何分配**。如果 AI 创造的财富只流向少数人，而被取代的工人没有得到适当支持，那么社会动荡是不可避免的。

这不仅是一个技术问题，更是一个政治和经济问题。
